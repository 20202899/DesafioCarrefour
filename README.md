# DesafioCarrefour

Aplicativo desenvolvido para teste Carrefour. 

Este app utiliza os seguintes patterns:
- Protocol Oriented Programming (POP)
- MVVM-C
- Factory Pattern
- ViewCode
- Coordinator
- SOLID
